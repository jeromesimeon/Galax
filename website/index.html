XML processing in O'Caml


What is Galax? Galax is an open-source implementation of XQuery, the W3C XML Query Language. It includes several advanced extensions for XML updates, scripting, and distributed programming.

Most of Galax's architecture is formally documented, making it ideal for users interested in teaching XQuery, in building new language extensions, or developing new optimizations.

Galax is implemented in O'Caml, a fast, modern, type-inferring, functional programming language descended from the ML (Meta Language) family.
 
Key Features.
  • XQuery 1.0 and XPath 2.0 support with 99.4% conformance.
  • XQuery type checking, modules, and XML schema import.
  • XQuery Updates, XQuery Scripting.
  • Web Services, and Distributed XQuery.
