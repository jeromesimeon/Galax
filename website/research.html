This page contains links to research papers and technical reports written at various point during the design of XQuery, as well as during the development of Galax. Those articles typically focus on hard technical problems, or important design issues.

Design and Semantics

  • XQuery From The Experts: A Guide to the W3C XML Query Language, Howard Katz, Don Chamberlin, Denise Draper, Mary Fernández, Michael Kay, Jonathan Robie, Michael Rys, Jérôme Siméon, Jim Tivy, Philip Wadler.
  • Growing XQuery. Mary Fernández and Jérôme Siméon. ECOOP'2003 (European Conference on Object-Oriented Programming).
  • XML Query Languages: Experiences and Examplars. Mary Fernández, Jérôme Siméon, Philip Wadler (editors). Communication to the XML Query W3C Working Group. September 1999.
  • An Algebra for XML Query. Mary Fernández, Jérôme Siméon, Philip Wadler. FST TCS, Delhi, December 2000.

Type System

  • The Essence of XML, Philip Wadler, and Jérôme Siméon. ACM Conference on Principles of Programming Languages (POPL'2003). January 2003.
  • A semistructured monad for semistructured data. Mary Fernández, Jérôme Siméon, Philip Wadler. ICDT, London, January 2001.
  • Subsumption for XML Types. Gabriel M. Kuper and Jérôme Siméon, in International Conference on Database Theory (ICDT'2001), January 2001, London, UK.

Optimization

  • XQuery Streaming à la Carte, Mary Fernández, Philippe Michiels, Jérôme Siméon, Michael Stark. ICDE 2007: 256-265.
  • Put a Tree Pattern in Your Algebra, Philippe Michiels, George A. Mihaila, Jérôme Siméon. ICDE 2007: 246-255.
  • How to Recognise Different Kinds of Tree Patterns From Quite a Long Way Away, Jan Hidders, Philippe Michiels, Jérôme Siméon. PLAN-X 2007: 14-24.
  • A Complete and Efficient Algebraic Compiler for XQuery., Christopher Re, Jérôme Siméon, Mary Fernández. ICDE 2006.
  • Optimizing Sorting and Duplicate Elimination in XQuery Path Expressions, Mary Fernández, Jan Hidders, Philippe Michiels, Jérôme Siméon, Roel Vercammen, DEXA 2005, 554-563.
  • Projecting XML Documents, Amélie Marian and Jérôme Siméon. VLDB'2003. Berlin, Germany, September 2003.

XML Storage

  • The Simplest XML Storage Manager Ever, Avinash Vyas, Mary Fernández, Jérôme Siméon, XIME-P 2004 Workshop, 37-42.
  • StatiX: making XML count, Juliana Freire, Jayant R. Haritsa, Maya Ramanath, Prasan Roy, Jérôme Siméon. ACM SIGMOD 2002: 181-191.
  • From XML Schema to Relations: A Cost-Based Approach to XML Storage, Philip Bohannon, Juliana Freire, Prasan Roy, Jérôme Siméon. ICDE 2002: 64-.

Web Services

  • XQuery at your web service, Nicola Onose, Jérôme Siméon. WWW'2004, New York, NY.

XML Updates

  • Commutativity Analysis for XML Updates, Giorgio Ghelli, Kristoffer Høgsbro Rose, Jérôme Siméon, ACM Transactions on Database Systems, 33(4):1-47, 2008.
  • A Better Semantics for XQuery with Side-Effects, Giorgio Ghelli, Nicola Onose, Kristoffer Høgsbro Rose, Jérôme Siméon. DBPL 2007: 81-96.
  • Commutativity Analysis in XML Update Languages, Giorgio Ghelli, Kristoffer Høgsbro Rose, Jérôme Siméon. ICDT 2007: 374-388.
  • XQuery!: An XML Query Language with Side Effects, Giorgio Ghelli, Christopher Re, Jérôme Siméon, DataX Workshop, 2006.

Distribution

  • Highly distributed XQuery with DXQ, Mary Fernández, Trevor Jim, Kristi Morton, Nicola Onose, Jérôme Siméon. SIGMOD 2007: 1159-1161.
  • DXQ: a distributed XQuery scripting language, Mary Fernández, Trevor Jim, Kristi Morton, Nicola Onose, Jérôme Siméon. XIME-P 2007.
